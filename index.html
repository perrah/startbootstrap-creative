Order Status Bot

The Order Status Bot is an application aimed at retail companies and businesses that send products to customers. Through the automated bot service customers can find out where their delivery is, request a change in delivery date and more.

Using Amazon Lex for Natural Language Understanding and Lambda for executing business logic all of this can be handled through customers communicating with the Facebook Messenger application

Tiny Rhino

Tiny Rhino was a small independent Web Development based in Bristol, UK. The small team made multiple ranging web solutions for small local businesses.

The technologies used ranged from Droopal, Angular, Bootsrap & Cloud Infrastructure. One solution created was a generalised e-commerce site that allowed users to update their own products list.

Crypto Tracker

The Crypto Tracker is a personal project tracking real time price of crypto currencies with the aim of monitoring and smart analtyical alerts.

The tracker was developed on top of the Python Flask framework to run the monitoring & analytical services, InfluxDB to store real time streaming data. All deployed to a Raspberry Pi.
